    Nodo

    Nodo de lista simplemente enlazada¶ ↑

    Contiene dos valores, value, que tiene el elemento que contiene el nodo, y next, que tiene dentro el siguiente elemento de la lista.

    Nodo2

    Nodo de la lista doblemente enlazada¶ ↑

    Contiende el simbolo :prev, que contiene el elemento previo, value que contiene el elemento del nodo, y next, que tiene dentro el siguiente elemento de la lista.
